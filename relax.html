Snakes.zip (If there is network problem, you can download them through Baidu Netdisk. 如果有网络问题，可通过百度网盘下载。)

This is a greedy snake game I developed using online resources based on Unity and C# during my internship. Give it a try.

Note:
1. The game is only available in Chinese, but due to its simple gameplay, players can quickly understand the gameplay without using their native language;
2. After downloading the zip installation package, unzip to obtain the game files, and open the Snake_new.exe to start the game;
3. There is no exit button on the main interface of the game. You can exit by pressing the Windows logo key and deleting it in the taskbar;
4. The historical record data of the game's main interface was generated by me during testing. I don't want to delete it, and you can try breaking this record;
5. There may be unknown bugs in the game. If you are willing, you can contact me, but I am unlikely to modify [manual humor] (due to busy schedule).

这是我在实习中利用网络资源基于Unity和C#开发的贪吃蛇小游戏，试着玩一下吧。

注意：
1.游戏只有中文版，不过由于玩法简单，不使用母语也能很快理解玩法；
2.下载zip安装包后，解压即可获得游戏文件，打开Snake_new.exe开始游戏；
3.游戏主界面没有退出按钮，可以通过按下Windows徽标键并在任务栏中删除来退出；
4.游戏主界面的历史记录数据是我在测试的时候打出来的，不想删了，你可以试试打破这个记录；
5.游戏可能有未知bug，如果你愿意可以联系我，不过我大概率不会修改【手动滑稽】（因为忙）。
