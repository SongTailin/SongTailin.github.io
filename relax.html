Greedy Snake 贪吃蛇

Snakes.zip (Due to Github's 100MB file size limit, please download through Baidu Netdisk. 由于Github有100MB文件大小限制，请通过百度网盘下载。)

Note:
1. The game is only available in Chinese, but due to its simple gameplay, players can quickly understand the gameplay without language;
2. After downloading the zip installation package, unzip to obtain the game files, and open the Snake_new.exe to start the game;
3. There is no exit button on the main interface of the game. You can exit by pressing the Windows logo key and deleting it in the taskbar;
4. The historical record data of the game's main interface was generated by me during testing. I don't want to delete it, and you can try breaking this record;
5. There may be unknown bugs in the game. If you are willing, you can contact me, but I am unlikely to modify(￢‿￢) (due to busy schedule).
注意：
1.游戏只有中文版，不过由于玩法简单，不用语言也能很快理解玩法；
2.下载zip安装包后，解压即可获得游戏文件，打开Snake_new.exe开始游戏；
3.游戏主界面没有退出按钮，可以通过按下Windows徽标键并在任务栏中删除来退出；
4.游戏主界面的历史记录数据是我在测试的时候打出来的，不想删了，你可以试试打破这个记录；
5.游戏可能有未知bug，如果你愿意可以联系我，不过我大概率不会修改(￢‿￢)（因为忙）。
